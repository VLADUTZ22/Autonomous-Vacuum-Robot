# Autonomous-Vacuum-Robot

## Overview
This project presents an autonomous vacuum robot developed using Arduino Uno, an ultrasonic sensor, and an electronically controlled motor system.

The robot is designed to navigate independently, detect obstacles, and avoid collisions using sensor-based distance measurements. A vacuum system was integrated with the robotic platform to simulate an automated cleaning process.

The project involved both hardware assembly and software programming, focusing on movement control, obstacle detection, motor control, and basic robotics concepts.

## Technologies Used
- Arduino Uno
- Embedded Systems
- Robotics
